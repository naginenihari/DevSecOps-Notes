 ansible all -i 98.80.13.36, -e ansible_user=ec2-user -e ansible_password=DevOps321 -b -m dnf -a "name=nginx status=installed"

playbook means list of tasks to be executed on the target server.

 ansible-playbook -i invertory.ini -e ansible_user=ec2-user -e ansible_password=DevOps321 main.yaml



single server creation:

ansible-playbook -i inventory.ini 01.1-EC2-Route53-Record.yaml -e "instances=['catalogue']"

ansible-playbook -i inventory.ini 01.1-EC2-Route53-Record.yaml -e "instances=['mongodb','catalogue','redis','user','cart','mysql','shipping','rabbitmq','payment','frontend']"
